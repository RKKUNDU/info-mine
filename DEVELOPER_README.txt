*.key is the files containing the encryption key for getting credentials of a particular service.
1. moodle.key
2. insti_mail.key
3. dept_mail.key

Similarly, there will be following files for storing encrypted (encryption key is present in the corresponding key file) credentials of the services
1. moodle
2. insti_mail
3. dept_mail

All the 6 files will be stored in configs/ folder (but don't push these files in the repository)
